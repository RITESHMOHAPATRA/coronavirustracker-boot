# Cronavirus Tracker
COVID-19 Tacker using Spring Framework

### Dataset
COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University

https://github.com/CSSEGISandData/COVID-19

### Tech Used
  - Java 11
  - Spring Boot
  - Thymleaf
  - HTML, CSS, Bootstap
  - Apache Mave
  - Heroku
